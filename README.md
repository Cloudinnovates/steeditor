## Quick setup

```
git clone https://github.com/jakipatryk/steeditor.git

cd steeditor

npm install
```

At this point edit the `src/environments/environment.ts` file. Create a SteemConnect app and fill out its details in `steemConnectConfig` object. Then:

```
npm start
```
